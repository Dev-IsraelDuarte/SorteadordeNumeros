## 📝 Sobre o Projeto
Este projeto foi criado para facilitar a realização de sorteios rápidos. O usuário define um intervalo (mínimo e máximo), e o sistema gera um número aleatório, mantendo um histórico dos resultados para conferência.

## 🚀 Funcionalidades
* **Sorteio Aleatório:** Gera números baseados no intervalo definido pelo usuário.
* **Histórico em Tempo Real:** Cada número sorteado é adicionado ao topo de uma lista.
* **Limpeza de Dados:** Botão para resetar o histórico e o resultado atual.
* **Validação de Intervalo:** Impede que o sorteio ocorra se o valor mínimo for maior que o máximo.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** C#
* **Framework:** .NET (Windows Forms)
* **IDE:** Visual Studio

## 💻 Como rodar a aplicação
1. Abra o arquivo `.sln` no **Visual Studio**.
2. Pressione `F5` ou clique em **Iniciar** para compilar e rodar.
